#   map
A simple map service

### 投影方式:
|名称|描述|
|----|----|
|MercatorProjection   |Web墨卡托投影 |
|LinearProjection     |线性投影      |
|GaussKrugerProjection|高斯-克里格投影|

### 在线地图:
|名称|地理坐标系|投影方式|备注|
|----|---------|-------|---|
|谷歌地图|WGS84|Web墨卡托投影|
|百度地图|WGS84|Web墨卡托投影|BD09坐标系|
|天地图  |WGS84|Web墨卡托投影/线性投影|
|高德地图|WGS84|Web墨卡托投影|
|苏州地图|Beijing54|高斯-克里格投影|iserver发布的服务，内网|
